# 写在前面

- 是什么？这是一个区块链学习资源库，收录区块链开发，区块链计算机理论，区块链与金融，区块链产业发展等内容

- 为什么创建？
  - 1. 目前很多区块链资源仓库很久没有进行更新了；
  - 2. 期望创立一个循序渐进的，而非简单罗列的区块链学习仓库
  
- 提供帮助？🧰 [点击](https://github.com/Eternaldeath/blockchainHome/issues) 🧰
